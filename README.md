# NFSU Türkçe Yama
Need for Speed Underground Türkçe Yama Çalışması Bu Türkçe Yama çalışması şahsıma aittir. Farklı kaynaklardan örnek alınmıştır fakat tamamen kendi bilgilerimle 0'dan yapılmıştır. Çeviriler tek tek değiştirilmiştir. Bununla birlikte Türkçe karakterlerin görselliği düzeltilerek tamamen doğal bir görünüm kazandırılmıştır.

<img width="1920" height="1080" alt="NFSU (1)" src="https://github.com/user-attachments/assets/8cd9bf4c-04b6-46c1-90d4-8f6ca9e6d959" />

Yamayı indirmek için 'Releases' sayfasına gidiniz.

Zip dosyasının şifresi 'Releases' sayfasındaki açıklamada belirtilmiştir.

Yama kurulumu için 'Kurulum Talimatları' text dosyasını okuyunuz.
